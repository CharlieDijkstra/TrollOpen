# TrollOpen
巨魔扇形悬浮，快捷启动app

## 功能描述：
1.扇形布局快捷启动app

2.更改悬浮颜色

3.已添加界面可长按排序app

4.图标大小（✅）

4.左手模式（✅）

5.震动反馈（✅）

6.显示触摸app（✅）

7.调节震动级别（✅）

8.快捷指令（✅）

9.搜索功能（✅）
